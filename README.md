# Abel Beltrán - Freelancer Portfolio

## Project Overview

This is a personal freelancer portfolio website for Abel Beltrán. The site showcases services in web design, mobile app development, and e-commerce solutions, and provides a contact form for potential clients.

## Features

- Responsive design for all devices
- Modern, clean layout
- Service descriptions with icons
- Contact form for inquiries
- Custom styles using CSS variables and utility classes

## Project Structure

```bash
freelancer/
├── index.html              # Main HTML file
├── css/
│   ├── normalize.css       # CSS reset for cross-browser consistency
│   └── styles.css          # Main custom styles
└── img/
    └── hero.jpg            # Hero section background image
```

## How to Use

1. **Clone or download the repository.**
2. Open `index.html` in your web browser to view the site.
3. To customize content, edit `index.html` and update styles in `css/styles.css`.

## Customization

- **Colors and Fonts:**
  - Colors are defined as CSS variables in `styles.css` for easy updates.
  - The Krub font is loaded from Google Fonts.
- **Images:**
  - Replace `img/hero.jpg` with your own image for a personalized hero section.
- **Services:**
  - Update the service sections in `index.html` to reflect your own offerings.
- **Contact Form:**
  - The form is static and does not send emails by default. Integrate with a backend or service (like Formspree) to enable submissions.

## Accessibility & Best Practices

- Semantic HTML is used for structure and clarity.
- All interactive elements are accessible via keyboard.
- Responsive design ensures usability on all devices.
