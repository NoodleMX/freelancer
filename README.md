# Freelancer Portfolio Website

A modern, responsive portfolio website for freelance services, featuring web design, mobile app development, and e-commerce solutions.

## Features

- Responsive navigation
- Hero section with location indicator
- Services showcase
- Contact form
- Scroll snap navigation
- Mobile-first design approach

## Technologies Used

- HTML5
- CSS3
- Custom CSS Variables
- CSS Grid
- CSS Flexbox
- Google Fonts (Krub)
- Normalize.css

## Project Structure

```plaintext
freelancer/
│
├── css/
│   ├── normalize.css    # CSS reset and normalization
│   └── styles.css      # Custom styles
│
├── img/
│   └── hero.jpg        # Hero section background
│
└── index.html          # Main HTML file
```

## Setup

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - static HTML/CSS only

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Features Breakdown

### Responsive Design

- Mobile-first approach
- Breakpoints at 480px and 768px
- Fluid typography with rem units

### CSS Features

- Custom properties (variables) for colors
- CSS Grid for services layout
- Flexbox for navigation and alignment
- Smooth scroll snapping
- Box-shadow effects
- Linear gradients for background

### Components

- Navigation bar with hover effects
- Hero section with overlay
- Service cards with icons
- Contact form with fieldset
- Footer
